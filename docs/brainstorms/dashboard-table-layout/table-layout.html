Notes on this layout

  • Status icons: 🟢 on / 🔴 off / ⚫ unavailable (plug off or publisher down)
  • Host name: shown only on first interface row, bold. Subsequent interfaces have blank host cell.
  • Host separator: thin border between hosts for readability
  • IPv6 suffix: common prefix (e.g. 2404:e80:a137:105) stripped, showing only ::150. Prefix shown once in the card header.
  • Stack column: abbreviated — "dual" / "v4" / "v6" / "unreach" / "unavail"
  • Down hosts: dimmed with opacity for visual de-emphasis
  • Controls column: shows which Tasmota plug powers this device (if any)
  • Location column: from spreadsheet Physical Location field
