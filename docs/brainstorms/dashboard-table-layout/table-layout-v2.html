Design summary

  • Clickable sort headers: "Host ▾" (gold = active sort), "St", "Stack", "RTT" (blue = clickable). Each links to a pre-sorted view.
  • 3 views generated: by-name (default), by-status (online first), by-rtt (fastest first). Same tables, different row order.
  • IPv6 suffix: common prefix shown in card header, only suffix (::NNN) in cells
  • Visual hierarchy: Online rows at full opacity, offline at 0.5, unavailable at 0.35
  • One card per network: net (41 hosts), int (5), iot (82), roam (20), pwr (3), etc.
  • Controls column: which Tasmota plug powers this device
